# FilenameEdit
edit multiple filenames at a time
os: win11  
ide: Qt 6.8 / Qt Creator
compiler: msvc 2022

> **Note:** Qt Creator user-specific files (`*.user`, `*.user.*`, `.qtcreator/`) are listed in `.gitignore` and are not tracked by this repository.

menu->addfile to add files
